Python Training
Important Rules
  • Python is indentation-sensitive
  • Dynamic typing (no need to declare types)
  • Everything is an object
Example

def greet(name):
    return f"Hello, {name}"

print(greet("Sasi"))

Python is widely used in web development, data science, automation, and AI.

SQL Training
Important Rules
  • SQL is case-insensitive
  • Use PRIMARY KEY for uniqueness
  • Normalize data to avoid duplication
Example

SELECT name, email
FROM users
WHERE is_active = 1
ORDER BY name;

SQL is used to manage and query relational databases like MySQL, PostgreSQL, and SQLite.

Django Training
Important Rules
  • MVT architecture (Model-View-Template)
  • DRY principle (Don't Repeat Yourself)
  • Use ORM instead of raw SQL
Example

# views.py
from django.shortcuts import render

def home(request):
    return render(request, "home.html")

Django is a high-level Python framework used to build secure and scalable web applications.
